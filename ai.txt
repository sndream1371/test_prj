question)통신사 청구에서 수납금과 납부금의 차이점을 알려주세요
통신사 청구에서 수납금과 납부금은 다음과 같은 차이점이 있습니다:

1. 수납금: 청구된 요금 중 고객이 실제로 지불한 금액을 의미합니다. 예를 들어, 통신사에서 청구한 요금이 10만원이지만, 고객이 8만원을 지불한 경우, 
           8만원이 수납금이 됩니다. 수납금은 고객이 통신사에 지불한 실제 금액을 나타냅니다.

2. 납부금: 청구된 요금 중 통신사가 고객으로부터 받은 금액을 의미합니다. 예를 들어, 통신사에서 청구한 요금이 10만원이지만, 
          고객이 8만원을 지불한 경우, 8만원이 납부금이 됩니다. 납부금은 통신사가 실제로 고객으로부터 받은 금액을 나타냅니다.

따라서, 수납금과 납부금의 차이는 통신사가 청구한 요금과 실제로 고객이 지불한 금액 사이의 차이를 나타냅니다.

오라클로 구성된 DB에서 아래정보를 정재해서 마이그데이터를 만들고 싶어
pk는 가입계약아이디 이며 입력하려는 정보는 가입계약아이디의 가입시작일, 가입종료일 이며
아래 3개의 테이블의 데이터를 이용하여, 타겟 테이블에 정보를 넣으려고해
가입 테이블 : 가입정보만 존재, 해지시 가입_해지 테이블로 레코드 이동
가입_해지 테이블 : 가입계약아이디, 가입시작일 정보 존재
가입_이력 테이블 : 가입계약아이디, 가입종료일 가입해지 정보를 가입계약아이디로 상태코드=C로 업데이터

타겟 테이블 : 가입계약아이디, 가입시작일, 가입종료일

=====================================>
아래는 오라클로 구성된 DB에서 마이그레이션 데이터를 만들기 위한 쿼리입니다.

먼저, 가입_해지 테이블에서 가입계약아이디와 가입시작일 정보를 가져옵니다.

```
INSERT INTO 타겟테이블 (가입계약아이디, 가입시작일)
SELECT 가입계약아이디, 가입시작일
FROM 가입_해지테이블;
```

그리고 가입_이력 테이블에서 가입계약아이디와 가입종료일을 가져와서 타겟테이블에 업데이트합니다.

```
UPDATE 타겟테이블
SET 가입종료일 = (
    SELECT 가입종료일
    FROM 가입_이력테이블
    WHERE 가입_이력테이블.가입계약아이디 = 타겟테이블.가입계약아이디
)
WHERE EXISTS (
    SELECT 1
    FROM 가입_이력테이블
    WHERE 가입_이력테이블.가입계약아이디 = 타겟테이블.가입계약아이디
);
```

이렇게 하면 타겟테이블에 가입계약아이디, 가입시작일, 가입종료일의 정보가 적재됩니다.
